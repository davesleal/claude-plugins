Overview

What It Does

OpenHAL 9000 gives Claude Code a voice. Every response is spoken aloud using macOS native text-to-speech, channeling the calm, measured tone of HAL 9000 from 2001: A Space Odyssey. No API keys. No cloud services. Just Claude, speaking to you from your terminal.

Capabilities

Features

> Native TTS

Uses macOS built-in speech synthesis. No API keys, no cloud dependency, no latency. Works completely offline.

> Zero Config

Install the plugin and HAL starts speaking. No setup wizard, no config files to edit, no environment variables to set.

> In Character

Every response is spoken in HAL's distinctive cadence. The calm, deliberate delivery that defined a generation of science fiction.

> Voice Control

Toggle HAL on and off, adjust modes, or trigger speech manually. Full control over when and how HAL speaks.

Get Started

Installation

1

Install the plugin

Add OpenHAL 9000 to your Claude Code setup:

claude plugin add openhal-9000
2

Run setup

Initialize the HAL voice system on your machine:

/hal setup
3

Start talking

HAL is now active. Every Claude response will be spoken aloud. Control it with:

/hal on # Enable voice /hal off # Disable voice /hal say "..." # Speak custom text /hal status # Check current state
