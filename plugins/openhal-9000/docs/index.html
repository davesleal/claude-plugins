Installation Sequence
01
Add to your plugin marketplace
{
  "plugins": {
    "openhal-9000": {
      "url": "github:daveleal/claude-plugins/openhal-9000"
    }
  }
}
02
Install the plugin
/plugins install openhal-9000
03
Start speaking

Once installed, Claude will speak every response aloud as HAL 9000. Use /hal off to silence, /hal on to resume, or /hal say "any text" to speak on demand.
