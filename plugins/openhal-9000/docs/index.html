System Briefing

A Claude Code plugin that gives your AI assistant a voice. Specifically, the voice of HAL 9000. Claude speaks every response aloud. He doesn't read text. He editorializes. Calm, precise, politely unsettling. Native Piper TTS on Apple Silicon. No Docker. No ffmpeg. Just Python and Claude Code.

Capabilities

Native TTS

Piper runs directly on Apple Silicon. No containers. No servers. No network overhead. ~0.7 seconds per sentence.

Zero Config

First session bootstraps everything automatically. Downloads model, creates venv, verifies install. HAL sings you a song.

In Character

HAL doesn't read your code aloud. He editorializes. Summarizes. Calm, precise, and politely unsettling.

Voice Control

Toggle voice on and off. Adjust speech rate. Full /hal command suite for mode, status, setup.
